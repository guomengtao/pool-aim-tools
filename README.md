# 台球瞄准辅助工具

透明台球瞄准叠加层工具，支持点击穿透和实时瞄准线绘制。

## 功能特点

- ✅ **点击穿透**：窗口外的点击会传递到下面的应用程序
- ✅ **实时瞄准线**：点击窗口内显示到6个球洞的虚线瞄准线
- ✅ **10秒自动清除**：瞄准线10秒后自动消失
- ✅ **竖版布局**：针对竖版台球桌优化（224×432）
- ✅ **半透明显示**：浅灰色虚线，不遮挡游戏画面
- ✅ **始终置顶**：窗口在所有应用之上

## 系统要求

- Python 3.7+
- PyQt5
- pynput

## 安装步骤

```bash
git clone https://github.com/guomengtao/pool-aim-tools.git
cd pool-aim-tools
pip install -r requirements.txt
```

## 使用方法

```bash
python final_overlay.py
```

## 操作说明

| 操作 | 说明 |
|------|------|
| 点击窗口内 | 放置瞄准点，显示到6个球洞的虚线 |
| 点击窗口外 | 点击穿透到台球游戏 |
| 按 `q` 键 | 退出程序 |

## 窗口位置

- 默认位置：距离左边 100 像素，垂直居中
- 尺寸：224×432（标准台球桌比例）

## 工作原理

1. 程序启动后，屏幕左侧出现透明覆盖层
2. 点击窗口内任意位置标记目标点
3. 显示从目标点到6个球洞的半透明虚线
4. 利用这些虚线辅助瞄准台球
5. 10秒后瞄准线自动消失
6. 窗口外的点击正常传递到台球游戏

## 注意事项

- 窗口完全透明，不会遮挡游戏画面
- 瞄准线为半透明浅灰色虚线，不影响查看台球
- 窗口始终置顶，在所有应用之上
- 按 `q` 键可随时退出程序
