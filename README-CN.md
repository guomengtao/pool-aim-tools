# Pool Aim Tools

台球瞄准辅助工具 - 透明覆盖层

## 功能特点

- ✅ **点击穿透**：窗口外的点击会穿透到下面的应用程序
- ✅ **实时瞄准线**：点击窗口内显示从点击点到6个球洞的连线
- ✅ **竖版台球桌布局**：优化竖版（纵向）台球桌方向
- ✅ **半透明显示**：浅灰色线条和圆圈，不遮挡游戏画面
- ✅ **始终置顶**：窗口保持在所有应用程序之上

## 系统要求

- Python 3.7+
- PyQt5
- pynput

## 安装步骤

### 1. 克隆仓库

```bash
git clone https://github.com/guomengtao/pool-aim-tools.git
cd pool-aim-tools
```

### 2. 创建虚拟环境（推荐）

```bash
python3 -m venv venv
source venv/bin/activate  # macOS / Linux
# 或在 Windows 上: venv\Scripts\activate
```

### 3. 安装依赖

```bash
pip install -r requirements.txt
```

## 使用方法

### 运行程序

```bash
python final_overlay.py
```

### 操作说明

| 操作 | 说明 |
|------|------|
| 点击窗口内区域 | 放置瞄准点，显示到6个球洞的连线 |
| 点击窗口外区域 | 点击穿透到台球游戏 |
| 按 `q` 键 | 退出程序 |

## 窗口位置

- 默认位置：距离左边 100 像素，垂直居中
- 尺寸：224px × 432px（标准台球桌比例）

## 工作原理

1. 透明覆盖层出现在屏幕左侧
2. 点击窗口内任意位置标记目标点
3. 从目标点到6个球洞绘制半透明连线
4. 使用这些连线可视化击球路径
5. 窗口外的点击正常传递到台球游戏

## 注意事项

- 覆盖层完全透明，不会干扰游戏
- 瞄准线为半透明（约40%透明度），避免遮挡球桌
- 窗口始终保持在所有应用程序之上
- 随时按 `q` 键关闭覆盖层
